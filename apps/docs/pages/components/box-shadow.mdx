# Box Shadow

In addition to the Tailwind's box shadow classes there are 7 specific box shadow classes provided by `@noqte/ui`. Use the `shadow-n-xs`, `shadow-n-sm`, `shadow-n-md`, `shadow-n-lg`, `shadow-n-xl`, `shadow-n-2xl`, `shadow-n-3xl` utilities to apply different sized box shadows.
